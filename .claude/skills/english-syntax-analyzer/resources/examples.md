# 실전 예시 (Examples)

수능 기출 및 주요 문장 유형별 구문분석 예시입니다.

---

## 1. 주어-동사 수일치 (2025 수능)

### 원문
> Patients recovering from a brain injury affecting the motor system knows that simple tasks can be extremely difficult.

### 구조도

```
[Patients (recovering from a brain injury) (affecting the motor system)] `knows`
 ────────  ───────────────────────────────  ──────────────────────────   ─────
    S                 현재분사구①                    현재분사구②            V
  (복수)                ↳ patients 수식                ↳ injury 수식     (단수❌)
```

### 성분 분석

| 기호 | 원문 | 성분 | 품사/기능 | 비고 |
|:----:|------|:----:|----------|------|
| S | Patients | 주어 | 명사(복수) | 핵심 주어 |
| ( )① | recovering from a brain injury | 수식어 | 현재분사구 | patients 수식 |
| ( )② | affecting the motor system | 수식어 | 현재분사구 | injury 수식 |
| V | knows → **know** | 동사 | 3인칭 단수 | ❌ 수일치 오류 |

### 문법 포인트

| 포인트 | 설명 |
|--------|------|
| **출제 유형** | 주어-동사 수일치 |
| **함정** | 긴 수식어구 사이에 핵심 주어 숨김 |
| **정답** | knows → know |

### 해설

> 핵심 주어는 Patients(복수)이다. recovering과 affecting은 모두 현재분사로
> 각각 앞의 명사를 수식하는 수식어구일 뿐이다. 따라서 동사는 복수형 know가 되어야 한다.

## 2. 관계대명사 vs 관계부사

### 원문
> This is the place where I was born.
> This is the place which I visited last year.

### 구조도 - where (관계부사)

```
This is [the place (where I was born)].
──── ──  ─────────  ─────────────────
  S   V      C           관계부사절
                         ↳ 완전한 문장
```

### 구조도 - which (관계대명사)

```
This is [the place (which I visited Ø last year)].
──── ──  ─────────  ────────────────────────────
  S   V      C              관계대명사절
                            ↳ 불완전 (목적어 없음)
```

### 비교 분석

| 구분 | where | which |
|------|-------|-------|
| 뒤 문장 | 완전 (S+V) | 불완전 (O 없음) |
| 역할 | 부사 (장소) | 대명사 (목적어) |
| 선택 기준 | 문장이 완전하면 | 문장이 불완전하면 |

### 판별법

```
where I was born     → I was born (완전) → where ✓
which I visited      → I visited Ø (불완전: 목적어 없음) → which ✓
```

## 3. 전치사 + 관계대명사 (2025년 5월 모의고사)

### 원문
> The extent to which scientists need to develop relationships is important.

### 구조도

```
[The extent (to which scientists need to develop relationships)] is important.
 ──────────  ─────────────────────────────────────────────────── ── ─────────
     S                        전치사+관계대명사절                   V     C
              ↳ to the extent에서 전치사 이동
```

### 성분 분석

| 기호 | 원문 | 성분 | 품사/기능 | 비고 |
|:----:|------|:----:|----------|------|
| S | The extent | 주어 | 명사구 | 선행사 |
| to which | to which | 전치사+관대 | 관계절 접속 | = to the extent |
| V | is | 동사 | be동사 | 2형식 |
| C | important | 보어 | 형용사 | 주격보어 |

### 문법 포인트

| 포인트 | 설명 |
|--------|------|
| **전치사 이동** | to the extent → the extent to which |
| **관계사 선택** | 전치사 뒤 → which (that 불가) |

## 4. 분사구문

### 원문
> Seen from a distance, the mountain looks beautiful.

### 구조도

```
(Seen from a distance), the mountain looks beautiful.
─────────────────────   ──────────── ───── ─────────
    분사구문(수동)           S          V       C
    ↳ = When it is seen
```

### 성분 분석

| 기호 | 원문 | 성분 | 품사/기능 | 비고 |
|:----:|------|:----:|----------|------|
| ( ) | Seen from a distance | 분사구문 | 부사적 수식 | 수동 (산이 보여짐) |
| S | the mountain | 주어 | 명사구 | 분사구문의 의미상 주어 |
| V | looks | 동사 | 감각동사 | 2형식 |
| C | beautiful | 보어 | 형용사 | 주격보어 |

### 분사구문 복원

```
Seen from a distance
= When it is seen from a distance
= 멀리서 보면
```

### 능동 vs 수동 판별

| 주어 | 분사 | 판단 |
|------|------|------|
| mountain | seen | 산이 보여짐 → 수동 → p.p. |
| I | seeing | 내가 봄 → 능동 → -ing |

## 5. 사역동사 + 목적격보어

### 원문
> I had my car repaired at the garage.

### 구조도

```
I had my car repaired (at the garage).
─ ─── ────── ──────── ──────────────
S  V    O      OC          M
       ↳ 차가 수리당함 (수동)
```

### 성분 분석

| 기호 | 원문 | 성분 | 품사/기능 | 비고 |
|:----:|------|:----:|----------|------|
| S | I | 주어 | 대명사 | |
| V | had | 동사 | 사역동사 | 5형식 |
| O | my car | 목적어 | 명사구 | |
| OC | repaired | 목적격보어 | 과거분사 | O가 수동 → p.p. |
| M | at the garage | 수식어 | 전치사구 | 장소 |

### 사역동사 보어 형태

| 구조 | 의미 | 예시 |
|------|------|------|
| have + O + 원형 | O가 ~하게 하다 | have him go |
| have + O + p.p. | O가 ~되게 하다 | have it done |

## 6. 가주어-진주어 구문

### 원문
> It is important that you study hard every day.

### 구조도

```
It is important [that you study hard every day].
── ── ───────── ─────────────────────────────────
가S  V     C              진주어(명사절)
```

### 성분 분석

| 기호 | 원문 | 성분 | 품사/기능 | 비고 |
|:----:|------|:----:|----------|------|
| It | It | 가주어 | 형식 주어 | 진주어를 대신 |
| V | is | 동사 | be동사 | 2형식 |
| C | important | 보어 | 형용사 | 주격보어 |
| [ ] | that you study... | 진주어 | 명사절 | 실제 주어 |

### 구조 복원

```
It is important that you study hard.
= That you study hard is important.
  (진주어가 앞으로 오면 문장이 머리무거움)
```

## 7. 강조 구문

### 원문
> It was yesterday that I met her at the park.

### 구조도

```
It was yesterday [that I met her at the park].
── ─── ───────── ──────────────────────────────
강조   강조대상              나머지 문장
```

### 강조 구문 판별

```
It is/was ~ that ... 구문에서:
- that 이하를 제거해도 문장 성립 → 강조 구문
- that 이하 없이 문장 불완전 → 가주어-진주어
```

### 비교

| 구문 | 예시 | 판별 |
|------|------|------|
| 강조 | It was **Tom** that broke the window. | Tom broke the window. ✓ |
| 가주어 | It is important **that** you study. | × (that절 필수) |

## 8. 병렬 구조

### 원문
> She likes swimming, hiking, and running.

### 구조도

```
She likes [swimming], [hiking], and [running].
─── ───── ──────────  ────────      ────────
 S    V      O①         O②           O③
             ↳ 모두 동명사 (병렬)
```

### 병렬 체크

| 요소 | 형태 | 일치 |
|------|------|:----:|
| swimming | 동명사 | ✓ |
| hiking | 동명사 | ✓ |
| running | 동명사 | ✓ |

### 오류 예시

```
❌ She likes swimming, hiking, and to run.
                                 ───────
                                 to부정사 (불일치!)
```

## 9. 도치 구문

### 원문
> Never have I seen such a beautiful sunset.

### 구조도

```
Never have I seen such a beautiful sunset.
───── ──── ─ ──── ────────────────────────
 부정   조V  S  V           O
 부사
 ↳ 부정어 도치 (조동사+주어)
```

### 정상 어순 복원

```
도치: Never have I seen...
정상: I have never seen...
```

### 도치 유형

| 유형 | 예시 | 설명 |
|------|------|------|
| 부정어 도치 | Never have I... | 부정어가 문두 |
| 장소 도치 | Here comes the bus. | 장소부사가 문두 |
| So/Neither 도치 | So do I. | 동의 표현 |

## 10. 복합관계사

### 원문
> Whatever you do, do your best.

### 구조도

```
[Whatever you do], do your best.
─────────────────  ── ─────────
    부사절(양보)     V     O
= No matter what
```

### 복합관계사 정리

| 복합관계사 | = | 역할 |
|-----------|---|------|
| whatever | no matter what | 양보 부사절 |
| whoever | no matter who | 양보 부사절 |
| however | no matter how | 양보 부사절 |
| whenever | no matter when | 양보 부사절 |
| wherever | no matter where | 양보 부사절 |

---

## Quick Reference: 구문 분석 순서

1. **동사 찾기** → 문장의 중심
2. **주어 찾기** → 동사 앞, 수식어 제외
3. **목적어/보어** → 동사 뒤 필수 성분
4. **수식어** → 괄호로 묶기
5. **절 파악** → 접속사/관계사 확인
6. **형식 판단** → 1~5형식 결정
